h1. Quick Start

h2. Requirements

The following setup should work:

* Mac OS X 10.6
* Rubygems version >= 1.3.7
* bundler gem
* Firefox >=3.5

*Important:* Your Firefox must be installed in @/Applications/Firefox.app@!

h2. Installation

1. Open _Terminal_ @/Applications/Utilities/Terminal.app@
2. Create a project folder:
<pre>
	mkdir -p /tmp/projects/workspace/
	cd /tmp/projects/workspace/
</pre>
3. Load examples from Github:
<pre>
	git clone git://github.com/KieranHayes/cucumber_with_selenium.git "start-cucumber"
</pre>
4. Go into the new folder
<pre>
	cd start-cucumber
</pre>
5. Install required ruby gems:
<pre>
	sudo gem update --system
	sudo gem install bundler
	sudo bundle install
</pre>
6. Run cucumber on the command line:
<pre>
	cucumber CUCUMBER_HOST=http://www.spiegel.de/
</pre>


h2. Usage

Modify the examples within @features/@ folder
<pre>
	mate .
</pre>
